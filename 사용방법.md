# 📖 Slack 메뉴 봇 사용 방법

## 🎯 가장 쉬운 방법: 버튼 사용 (추천!)

### 방법 1: 봇 멘션하기
1. 채널에서 봇을 멘션합니다: `@봇이름`
2. 또는 "메뉴" 또는 "급식"이라고 입력합니다
3. **버튼이 나타납니다!** 🎉
4. 원하는 버튼을 클릭하세요:
   - 🌅 **오늘 메뉴** - 오늘의 급식 메뉴
   - 📅 **내일 메뉴** - 내일의 급식 메뉴
   - 📆 **모레 메뉴** - 모레의 급식 메뉴

### 방법 2: 슬래시 커맨드 사용
1. 채널에서 `/menu` 또는 `/메뉴` 입력
2. 엔터를 누르면 **버튼이 나타납니다!** 🎉
3. 원하는 버튼을 클릭하세요

---

## 📝 텍스트로 직접 요청하기

### 방법 3: 특정 키워드 입력
봇이 있는 채널에서 다음 중 하나를 입력하면 바로 오늘 메뉴가 표시됩니다:
- `오늘 메뉴`
- `오늘 급식`
- `today menu`
- `menu`

### 방법 4: 슬래시 커맨드로 날짜 지정
- `/menu` - 오늘 메뉴 (또는 버튼 표시)
- `/menu 내일` - 내일 메뉴
- `/menu 모레` - 모레 메뉴
- `/menu 2024-12-20` - 특정 날짜 메뉴 (YYYY-MM-DD 형식)

---

## 💡 사용 예시

### 예시 1: 버튼 사용 (가장 쉬움!)
```
사용자: @메뉴봇
봇: 🍽️ 어떤 메뉴를 확인하시겠어요?
     [🌅 오늘 메뉴] [📅 내일 메뉴] [📆 모레 메뉴]
     
사용자: [🌅 오늘 메뉴] 버튼 클릭
봇: *2025년 12월 19일 (금요일) 급식 메뉴*
     ...
```

### 예시 2: 텍스트로 직접 요청
```
사용자: 오늘 메뉴
봇: *2025년 12월 19일 (금요일) 급식 메뉴*
     ...
```

### 예시 3: 슬래시 커맨드
```
사용자: /menu 내일
봇: *2025년 12월 20일 (토요일) 급식 메뉴*
     ...
```

## 🎨 추천 사용법

**가장 추천하는 방법은 버튼 사용입니다!** 
- 클릭 한 번으로 원하는 메뉴를 볼 수 있어요
- 날짜를 기억할 필요가 없어요
- 직관적이고 사용하기 쉬워요

---

## ❓ 자주 묻는 질문

**Q: 봇이 응답하지 않아요**
A: 봇이 채널에 초대되어 있는지 확인하세요. 또는 DM으로 시도해보세요.

**Q: 버튼이 안 보여요**
A: `/menu` 명령어를 사용하거나 봇을 멘션해보세요.

**Q: 특정 날짜의 메뉴를 보고 싶어요**
A: `/menu 2024-12-20` 형식으로 날짜를 입력하세요.
